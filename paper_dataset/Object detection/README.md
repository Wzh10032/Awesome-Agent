### 📄 [Seq-NMS for Video Object Detection](http://arxiv.org/pdf/1602.08465v3)

👤 **Authors**: Wei Han, Pooya Khorrami, Tom Le Paine, Prajit Ramachandran, Mohammad Babaeizadeh, Honghui Shi, Jianan Li, Shuicheng Yan, Thomas S. Huang

📅 **Published**: `Feb 26, 2016` | 🔄 **Updated**: `Aug 22, 2016` | 🆔 **Arxiv ID**: [1602.08465v3](http://arxiv.org/pdf/1602.08465v3)

🤖 **AI Summary**:
> This paper proposes Seq-NMS, a modification of the post-processing phase in video object detection that uses detections from nearby frames to boost weaker detections within the same clip.
